# 用户使用说明书（非技术版）

## A. 部署前准备
1. 一台 Windows 电脑（作为主持人主机）。
2. 同事和你在同一个局域网/Wi-Fi。
3. 安装 Python（如已安装可跳过）。
4. 可选：准备 OpenAI API Key（用于 AI 翻译和高质量纪要）。

## B. 如何启动程序
1. 双击项目里的 `run.bat`。
2. 程序会自动打开浏览器进入主持人页面。
3. 如果没有自动打开，请手动访问：`http://127.0.0.1:8000/`。

## C. 如何召开会议
1. 主持人页面可看到会议码、二维码、局域网地址。
2. 参会者扫码或打开链接进入加入页。
3. 输入会议码、昵称、语言后点击“加入会议”。
4. 在主持人页面可切换：
   - 开启双向翻译（中文↔日语）
   - 不翻译（仅显示原文）

## D. 如何生成会议纪要
1. 在主持人页面点击“生成会议纪要”。
2. 等待几秒到十几秒。
3. 在页面预览后可下载 MD 或 TXT。

## E. 如何结束会议
1. 主持人点击“解散会议室”。
2. 所有人会议结束，不能继续发言或加入。
3. 系统会删除聊天记录、翻译结果、纪要和临时文件。

## F. 常见问题
- 扫二维码打不开：确认在同一 Wi-Fi，且主持人电脑未休眠。
- 局域网访问不到：检查防火墙是否允许 Python 网络访问。
- OpenAI 报错：检查 `.env` 的 API Key 和网络。
- 翻译失败：原文会保留，可继续开会。
- 修改端口：编辑 `.env` 里的 `PORT`，重启程序。
